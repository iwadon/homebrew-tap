# Iwadon Tap

## How do I install these formulae?

`brew install iwadon/tap/<formula>`

Or `brew tap iwadon/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "iwadon/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
